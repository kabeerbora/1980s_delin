# 1980s Delicensing Analysis

This repository contains analysis notebooks and scripts for studying delicensing patterns in India during the 1980s using Annual Survey of Industries (ASI) firm data.

## Repository Structure

- `1980s_delicensing.ipynb` - Main analysis notebook for delicensing patterns
- `Panel_79_89_v2.ipynb` - Panel data analysis for 1979-1989
- `delin_extraction.ipynb` - Data extraction and preprocessing
- `visualizations_delin.ipynb` - Visualization scripts for delicensing data
- `delicensed_script.R` - R script for statistical analysis
